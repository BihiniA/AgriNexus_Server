
# AgriGuard - Backend (Node.js with Express.js)

## Project Setup

1. Clone the repository** to your local machine.

2. Navigate to the server folder:
   bash
   cd server

3. Install dependencies:

   bash
   npm install


4. Start the server:

   node index.js


5. The server will run and listen for requests.
